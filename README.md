- **HTML5**: Semantic structure and layout.

## 💻 How to Run the Portfolio Locally

Since this is a static website built with pure HTML, CSS, and JS, you don't need any complex setup to run it! You have a few options:

### Option 1: Simply Open the File (Easiest)
1. Download or clone this repository to your computer.
2. Open the folder containing the files.
3. Double-click on the `index.html` file. It will automatically open in your default web browser.

### Option 2: Using VS Code Live Server (Best for Development)
1. Open the project folder in **Visual Studio Code**.
2. Go to the Extensions tab (Ctrl+Shift+X) and install the **Live Server** extension by Ritwick Dey.
3. Right-click on the `index.html` file in your file explorer.
4. Select **"Open with Live Server"**.
5. Your default browser will open the site automatically (usually at `http://127.0.0.1:5500`), and it will auto-reload if you make any changes to the code.

### Option 3: Python Local Server
If you have Python installed on your computer, you can run a simple local server:
1. Open your terminal (Command Prompt or PowerShell).
2. Navigate to the project directory: `cd path/to/portfolio`
3. Run the following command:
   ```bash
   python -m http.server 8000
   ```
4. Open your web browser and go to `http://localhost:8000`

## 🌐 Hosting Live on GitHub Pages
To make this website visible to the world for free:
1. Go to your repository settings on GitHub.
2. Click on **Pages** in the left sidebar.
3. Under "Build and deployment", set the **Source** to "Deploy from a branch".
4. Under **Branch**, select `main` and save.
5. In a few minutes, your site will be live at `https://jerrapallavi-lang.github.io/portfolio/`!
